# Client Churn & Brokerage Revenue Analysis

## 📊 Overview
This project analyzes client churn and brokerage revenue for a stock broking firm.

## 🎯 Objectives
- Identify churn risk
- Analyze revenue contribution
- Provide business recommendations

## 📂 Dataset
Includes client, trading, portfolio, and engagement data.

## 📈 Dashboard Features
- Churn analysis
- Revenue trends
- Risk scoring

## 💡 Key Insights
- Top clients contribute majority of revenue
- Low engagement leads to churn
- Revenue declines before churn

## 🧠 Business Impact
Helps identify high-risk clients and reduce revenue loss.

## 🛠 Tools Used
- Power BI
- Excel
- DAX
